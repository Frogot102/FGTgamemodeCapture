Привет. Это мой мод для командных захватов с экономикой, классами
и магазином под Superb Warfare. Ниже — всё, что нужно знать, если
ты админ сервера или просто хочешь разобраться, как тут всё работает.


── ЧТО ЭТО ВООБЩЕ ─────────────────────────────────────────────

Короче, идея простая: две (или больше) команды через ванильный
скорборд, точки захвата, базы, деньги за убийства и удержание зон,
магазин с оружием/техникой и классы (штурм, медик, техник, развед).

Матч не стартует сам — его запускает админ. Пока матч не идёт, можно
спокойно выбирать команду и класс. Когда матч активен — без команды
тебя кинет в спектатор.


── КЛАВИШИ (можно переназначить в управлении) ───────────────────

  J — выбор команды
  K — выбор класса
  B — магазин (только на своей базе!)
  M — меню матча (счёт, деньги, статистика после конца)

В меню матча (M) внизу есть кнопки для связи:
  • «Фидбек (Telegram)» — личка @frogot_1
  • «Канал Rubickhouse» — t.me/Rubickhouse1

Пишите баги, баланс, идеи — буду рад.


── ОСНОВНЫЕ МЕХАНИКИ ────────────────────────────────────────────

ЗАХВАТ ТОЧЕК
  Админ создаёт точки командой. Игроки стоят в радиусе — точка
  нейтрализуется, потом захватывается. Счёт идёт до победного
  значения (по умолчанию настраивается).

БАЗЫ
  У каждой команды своя база (зона). На базе — магазин, баффы.
  С базы нельзя просто так уйти и покупать где попало.

ЭКОНОМИКА
  • Личные деньги — на себя (оружие, броня, гаджеты)
  • Командные — на команду, тратит только капитан
  Деньги за киллы, удержание точек, захваты. Капитана назначает админ.

КЛАССЫ
  Штурмовик, медик, техник, разведчик — у каждого свой набор в
  магазине и стартовый кит при начале матча.

МАГАЗИН
  Личный и командный таб. Техника продаётся контейнерами Superb
  Warfare (как в креативе — красная/белая коробка). Только капитан,
  только на базе, только своя фракция:
    ВСРФ: T-90, БМП-2, Mi-28
    NATO: Little Bird, Abrams, Bradley
  Наведи на лот — в подсказке написано, какие снаряды брать.

ДРУГОЕ
  • Friendly fire выключен между союзниками
  • Падение бьёт сильнее (x2)
  • Респawn с кулдауном после смерти в матче
  • Осадный инструмент ломает защищённые блоки базы


── КОМАНДЫ (нужны права оператора / perm 2, кроме money/menu/shop) ─

Главный префикс:  /FGTgmC

Игрокам:
  /FGTgmC money          — баланс
  /FGTgmC menu           — открыть меню
  /FGTgmC shop buy <id>  — купить из консоли

Матч:
  /FGTgmC start          — старт с отсчётом
  /FGTgmC start now      — мгновенный старт
  /FGTgmC stop           — остановить
  /FGTgmC score          — таблица очков
  /FGTgmC reset          — сброс счёта
  /FGTgmC setwinscore <n> — очков до победы

Точки захвата:
  /FGTgmC create <имя> <радиус>
  /FGTgmC remove <имя>
  /FGTgmC list
  /FGTgmC setradius <имя> <радиус>

Базы:
  /FGTgmC base create <имя> <команда> <радиус>
  /FGTgmC base remove <имя>
  /FGTgmC base list
  /FGTgmC base setradius <имя> <радиус>
  /FGTgmC base particles on|off <имя>

Капитан:
  /FGTgmC captain set <команда> <игрок>
  /FGTgmC captain clear <команда>

Экономика:
  /FGTgmC economy set kill personal|team <сумма>
  /FGTgmC economy set hold personal|team <сумма>
  /FGTgmC economy give|take personal|team ...

Магазин:
  /FGTgmC shop reload              — перезагрузить пресет sbw_balance
  /FGTgmC shop preset load sbw_balance

Пресеты (см. ниже):
  /FGTgmC preset save <имя>
  /FGTgmC preset load <имя>
  /FGTgmC preset load <имя> merge
  /FGTgmC preset files list


── ПРЕСЕТЫ — КАК СОХРАНЯТЬ И ПОДГРУЖАТЬ КАРТУ ───────────────────

1. Построй на карте всё нужное: точки, базы, команды скорборда,
   настрой экономику как хочешь.

2. Сохрани:
     /FGTgmC preset save test1

   Файл ляжет сюда:
     .minecraft/zovcapture/presets/test1.dat
   (в Modrinth — папка профиля/instance)

3. Чтобы загрузить на сервере / в одиночке:
     /FGTgmC preset load test1

   Это заменит точки, базы, команды, магазин и экономику из пресета.
   merge — только добавит/обновит, не удаляя лишнее.

4. После load магазин сам обновится. Если что-то не так:
     /FGTgmC shop reload

Совет: сделай test1 как «эталон» карты, test2 для экспериментов.


── ТЕХНИКА И СНАРЯДЫ (шпаргалка) ───────────────────────────────

ВСРФ (командный магазин, капитан):
  T-90      → мелкокалиберные снаряды, управляемые ракеты
  БМП-2     → + винтовочные патроны (коробки)
  Mi-28     → + фугасные/бронебойные ракеты

NATO:
  Little Bird → мелкокалиберные снаряды + мелкокалиберные ракеты
  Abrams      → мелкокалиберные + Javelin
  Bradley     → мелкокалиберные + TOW + винтовочные

Снаряды тоже в командном магазине, с пометкой [ВSRF] или [NATO].


── ЗАВИСИМОСТИ МОДПАКА ─────────────────────────────────────────

Мод рассчитан на NeoForge 1.21.1 и модпак с Superb Warfare,
AshVehicle (для Abrams/T-90/Bradley), Wararmbise, NVG и т.д.
Без SBW магазин техники просто не выдаст контейнеры.
